# Family link 

## What is this?
An app that connects family members by sharing live locations and staying in touch.

## Phase 1 Features
- Child phone shars location
- Parent sees live location on a map
- Distance between Family memers is shown

## tech stack

-HTML, CSS, Javascript (Frontend) 
-node.js + Express (Backend)
-Firebase (real-time location)
-Google map API(Maps & Location)

## How to run it 

1.Clone the repository
2.Run npm install to install dependencies
3.Add your Firebase and Google Maps API keys to a .env file
4.Run node server.js to start the backend
5.Open index.html in your browser
